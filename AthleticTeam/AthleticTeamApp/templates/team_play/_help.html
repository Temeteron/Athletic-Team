{% if mode == 'edit' %}

This is the edit mode!

1. With 'Next' Button you can see next movements/pass
2. With 'Previous' Button you can return to the previous stage of Team Play
3. With 'Move Player' Button you can change player's start position (only if its the 1st movement)
4. With 'Move Ball' Button you can change ball position (when you change possession of the ball you clear all passes)
5. With 'Draw Player's Path' Button you can draw player's next movement (when you redraw path you clear all player's next paths)
6. With 'Draw Pass' Button you can draw next pass (when you redraw pass you clear all next ball passes)
7. With 'Delete Path' Button you can delete Player's path (on delete you clear all player's next paths)
8. With 'Delete Pass' Button you can delete next pass (on delete you clear all next ball passes
{% else %}

This is the view mode!

1. With 'Next' Button you can see next movements/pass
2. With 'Previous' Button you can return to the previous stage of Team Play
{% endif %}
